# Glassmorphism Generator

Микро-приложение для генерации CSS кода матового стекла (glassmorphism).

## Фичи
- Настройка прозрачности
- Настройка размытия (blur)
- Моментальный предпросмотр
- Генерация чистого CSS без мусора

## Зачем это нужно?
Надоело каждый раз подбирать значения `backdrop-filter` и `rgba` в инструментах разработчика. Сделал этот тул, чтобы быстро копировать готовый CSS-класс.
